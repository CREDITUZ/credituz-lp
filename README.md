# Credituz · Landing Page

Site institucional da Credituz — o sistema operacional e financeiro do setor imobiliário, com Inteligência Artificial.

## Estrutura

- `index.html` — landing page principal
- `pages/` — páginas internas (incorporadoras, imobiliárias, integrações, blog, legal)
- `assets/banks/` — logos de bancos parceiros (carrossel)
- `assets/backers/` — logos de parceiros tecnológicos e certificações
- `pages/_base.css` — design system compartilhado entre páginas internas
- `vercel.json` — configuração de hosting

## Tecnologia

HTML estático com CSS e JS vanilla. Sem build step, sem framework. Fontes: Fraunces, Geist, JetBrains Mono, Raleway (via Google Fonts).

## Deploy

Hospedado em Vercel. Cada push para a branch `main` dispara redeploy automático.
